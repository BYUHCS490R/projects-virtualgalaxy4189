Internet

The internet is a bunch of connected hardware: cables, routers etc and also a connected network.

Web

It is a linked and connected "web" of websites, files, documents, resources etc. which are connected via browsers.

How are the Internet and Web different?

The internet is the physical part and the stuff that runs those physical things whereas the Web is the inside, the juicy fruit, the information.

ASCII and Unicode

ASCII is a language that a computer uses to know specific letters or characters you want to use using numbers. Unicode is a way to show text from every language using numbers.

What is UTF-8?

It's the most common way to store Unicode text so computers can read it.

URL

The location/address of a website.

Domain Name

The name of a website.

HTTP

Rules for how websites get to your browser.

HTTPS

A secure HTTP to protect your data.

IP Address

A number to identify a device on the internet.

DNS

A system to change website names to IP addresses.

W3C Recommendations

Official rules for how the web works.

XML, XHTML, HTML, HTML5

XML is a data language, HTML is a website language, XHTML is a stricter version of html that also follows XML rules and HTML5 is the latest html language that has more things it can do.
