  1. An inline is within the specific tag you want to change, an embedded is within the html page in the header but not in the specific tag you want to change, and the external is a css style page that is linked to the html page and isn't directly in the html page.
  2. Any external editors cannot override the inline or embedded css. Also, external css lets you use the same styling for multiple pages with minimal coding whereas the others requires lots of additional code.
  3. A selector is used to specify a specific aspect or tag you want to change using css.
